vConestoga : vConestoga Test Environment

Created by Jonathan Bezeau on May 17, 2024

We need to host a second instance of vConestoga to demonstrate that we can host many separate vConestoga instances, test-deploy code updates from Student Portal Liquid Cinema, and test our own development skills for UI updates and features in the vConestoga code.

Components

  • Domain name staging.vconestoga.com

  • Set up feathers server

    • node, npm

    • nginx

    • pm2

    • pull jkxr-be code

    • pull vconestoga code and copy over jkxr-be portal

  • Set up colyseus server

    • node, npm

    • pm2

    • pull jkxr-colyseus-be code

  • Set up mongo server

    • mongodb

    • create admin account

    • run jkxr script (from feathers instance) to populate collections

  • Use same AntMedia instance as production vConestoga

  • Configure .env variables
