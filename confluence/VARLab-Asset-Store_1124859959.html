Project Name

VARLab Asset Store

Description

Browse and download art assets

  • cloud-hosted web server

  • thumbnail preview and download link for each art asset in the store

  • LTI authentication so we can link students to assets through course shell

vConestoga Team Responsibilities

  • Server creation and maintenance https://assetstore.vconestoga.com/

  • Storage of assets we already possess (and create/buy going forward)

  • Metadata about those assets (where we got them, geometry, size, artist etc)

  • UI/UX for adding items to the storage (uploading, editing, deleting files, creating metadata, processing)

  • Front-end search UI/UX that searches entirely within this repository and meets the special needs of the storage system

  • Create an API for searching that allows data to feed into the Meta Search from the Asset Team Bravo

  • LTI integration with eConestoga

  • Authentication with Microsoft

Asset Team Bravo Responsibilities

  • Served off a different subdomain of vConestoga.com (for now) with the support of vConestoga team

  • No internal storage of files or metadata. All realtime.

  • Interfaces with multiple other asset stores (as well as our own storage API from above)

  • Responsible for creating a unified search interface UI that abstracts and normalizes the information from multiple sources with a focus on decision-making.. not discovery.

  • Responsible for translating store/engine-specific terms into a unified view (if required) and enabling developers success in finding assets without special art knowledge.

  • Consumes the API from vConestoga Asset Storage team.

  • Applies a heuristic to the blended results, putting the most likely useful things at the top of the list.
